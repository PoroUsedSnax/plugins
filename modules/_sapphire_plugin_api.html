ContentTypeParameter

ContentTypeParameter: `; ${string}=${string}`
  • Defined in packages/api/src/lib/structures/http/Server.ts:153

RFC 1341 4: Defines a Content-Type’s parameter, which follows the following structure:

  • parameter = attribute “=” value
  • attribute = token
  • value = token / quoted-string
  • token = 1*<any CHAR except SPACE, CTLs, or tspecials>
  • tspecials = ( | ) | < | > | @ | , | ; | : | \ | " | / | [ | ] | ? | . | =
note

tspecials must be in quoted-string, to use within parameter values.

note

The definition of tspecials is the same as the RFC 822 definition of specials with the addition of the three characters /, ?, and =.

since

1.3.0
